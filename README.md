# Marketing Funnel & Conversion Analysis

## 📊 Overview
This project analyzes the marketing funnel to identify conversion drop-offs and improve customer conversion rates.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## 📈 Analysis Done
- Funnel stage analysis
- Conversion rate calculation
- Drop-off identification
- Performance visualization

## 🔍 Key Insights
- High drop-off in early stages
- Conversion decreases across funnel
- Only a small % becomes customers

## 💡 Recommendations
- Improve landing page conversion
- Optimize lead quality
- Better follow-up strategies
- Focus on high-performing channels
